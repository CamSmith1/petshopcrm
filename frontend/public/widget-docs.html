Dog Services Widget - Integration Guide

This guide explains how to integrate the Dog Services booking widget into your website. The widget allows your customers to book dog services directly from your website without being redirected to another page.

Quick Start

Add the following code to your website where you want the booking widget to appear:

<!-- Dog Services Widget -->
<div id="dog-services-widget"></div>
<script>
  (function(d, s, id) {
    var js, fjs = d.getElementsByTagName(s)[0];
    if (d.getElementById(id)) return;
    js = d.createElement(s); js.id = id;
    js.src = "https://api.dogservices.com/widget.js";
    js.setAttribute('data-api-key', 'YOUR_API_KEY');
    fjs.parentNode.insertBefore(js, fjs);
  }(document, 'script', 'dog-services-widget-js'));
</script>
<!-- End Dog Services Widget -->
Note: Replace YOUR_API_KEY with the API key provided to you after registering as a service provider.

Getting Started

1. Create a Service Provider Account

Before integrating the widget, you need to create a service provider account at dogservices.com/register.

2. Generate API Key

After creating an account, go to your dashboard and navigate to "Integration" section. Click "Generate API Key" and save both the API key and API secret.

3. Add Widget to Your Website

Copy the code snippet provided in the Quick Start section and add it to your website. You can place it anywhere you want the booking widget to appear.

Customization Options

The widget supports several customization options to match your website's look and feel.

Appearance

You can customize the widget's appearance by adding data attributes to the script tag:

<script 
  src="https://api.dogservices.com/widget.js"
  data-api-key="YOUR_API_KEY"
  data-color="#FF5733"
  data-text-color="#333333"
  data-font-family="Arial, sans-serif"
  data-border-radius="8px"
  id="dog-services-widget-js"></script>
Attribute Description Default
data-color Primary color for buttons and headings #4a90e2
data-text-color Text color #333333
data-font-family Font family Arial, sans-serif
data-border-radius Border radius for buttons and cards 8px

Service Filtering

You can filter the services displayed in the widget by adding service filters:

<script 
  src="https://api.dogservices.com/widget.js"
  data-api-key="YOUR_API_KEY"
  data-service-category="grooming,walking"
  id="dog-services-widget-js"></script>

This will only show services in the "grooming" and "walking" categories. Available categories: grooming, boarding, daycare, training, walking, sitting, other.

Security Considerations

Important: Never include your API secret in client-side code. The API secret should only be used in server-to-server communications.

Server-Side Integration

For enhanced security, you can implement a server-side component to generate a valid signature for widget requests: 

// Node.js example
const crypto = require('crypto');

function generateWidgetSignature(apiSecret, payload) {
  return crypto
    .createHmac('sha256', apiSecret)
    .update(JSON.stringify(payload))
    .digest('hex');
}

// Example payload
const payload = {
  timestamp: Date.now(),
  origin: 'https://yourwebsite.com'
};

const signature = generateWidgetSignature('YOUR_API_SECRET', payload);

// Now you can pass this signature to your frontend

This approach prevents tampering with widget requests and ensures that only your website can make valid booking requests.

API Reference

The widget communicates with the Dog Services API to fetch available services, check availability, and create bookings.

Available Endpoints

Endpoint Method Description
/api/widget/token POST Get a session token for widget initialization
/api/widget/services GET Get available services for the provider
/api/widget/availability GET Check availability for a specific service and date
/api/widget/booking POST Create a new booking

FAQ

Can I use multiple widgets on the same page?

Yes, you can have multiple widgets on the same page by using different container IDs. Make sure each widget has a unique container ID.

Does the widget work on mobile devices?

Yes, the widget is fully responsive and works on all devices including mobile phones and tablets.

Can I customize the services displayed in the widget?

Yes, you can specify which services to display using the data-service-category attribute or by setting up service filtering in your dashboard.

How are bookings confirmed?

When a customer makes a booking through the widget, both the customer and you (the service provider) will receive email confirmations. You can also see and manage all bookings in your dashboard.

Can I customize the booking flow?

Yes, you can customize certain aspects of the booking flow through your dashboard settings. For more advanced customizations, please contact our support team.
