BookingPro Widget - Integration Guide

This guide explains how to integrate the BookingPro booking widget into your existing business website. The widget allows your customers to book services directly from your website without being redirected to another platform.

Quick Start

Add the following code to your website where you want the booking widget to appear:

<!-- BookingPro Widget -->
<div id="bookingpro-widget"></div>
<script>
  (function(d, s, id) {
    var js, fjs = d.getElementsByTagName(s)[0];
    if (d.getElementById(id)) return;
    js = d.createElement(s); js.id = id;
    js.src = "https://api.bookingpro.io/widget.js";
    js.setAttribute('data-api-key', 'YOUR_API_KEY');
    fjs.parentNode.insertBefore(js, fjs);
  }(document, 'script', 'bookingpro-widget-js'));
</script>
<!-- End BookingPro Widget -->
Note: Replace YOUR_API_KEY with the API key provided in your Business Dashboard.

Getting Started

1. Create a Business Account

Before integrating the widget, you need to create a business account at bookingpro.io/register.

2. Set Up Your Services

After creating an account, go to your dashboard and set up your services, availability, and booking options. These services will be displayed in your booking widget.

3. Generate API Key

Navigate to the "Integration" section of your dashboard. Click "Generate API Key" and save your API key securely.

4. Add Widget to Your Website

Copy the code snippet provided in the Quick Start section and add it to your website. You can place it anywhere you want the booking widget to appear.

Customization Options

The widget supports extensive customization options to match your website's branding and design.

Appearance

You can customize the widget's appearance by adding data attributes to the script tag:

<script 
  src="https://api.bookingpro.io/widget.js"
  data-api-key="YOUR_API_KEY"
  data-primary-color="#2563eb"
  data-secondary-color="#4f46e5"
  data-text-color="#1e293b"
  data-font-family="Inter, sans-serif"
  data-border-radius="8px"
  data-layout="standard"
  id="bookingpro-widget-js"></script>
Attribute Description Default
data-primary-color Primary color for buttons and interactive elements #2563eb
data-secondary-color Secondary color for highlights and accents #4f46e5
data-text-color Main text color #1e293b
data-font-family Font family (uses web-safe fonts or Google Fonts) system-ui, sans-serif
data-border-radius Border radius for cards, buttons, and inputs 6px
data-layout Widget layout (standard, compact, expanded) standard

Content Filtering

You can filter the services displayed in the widget:

<script 
  src="https://api.bookingpro.io/widget.js"
  data-api-key="YOUR_API_KEY"
  data-service-category="consultation,treatment"
  data-staff-id="12345,67890"
  id="bookingpro-widget-js"></script>

This will only show services in the specified categories and/or offered by the specified staff members.

Advanced Customization

For more advanced customization options, you can use the BookingPro Dashboard to:

  • Create custom CSS themes
  • Configure custom form fields for each service
  • Set up pre-filled booking information
  • Customize confirmation emails and notifications
  • Define special hours, holidays, and buffer times

Security Considerations

Important: Your API key is unique to your business and should be kept secure. While it's safe to use in the widget code snippet, never share it publicly or expose it in unrelated areas.

Domain Restrictions

For enhanced security, you can restrict your API key to specific domains in your dashboard settings. This prevents unauthorized use of your widget on other websites.

API Reference

The widget communicates with the BookingPro API to fetch available services, check availability, and create bookings.

Available Endpoints

Endpoint Method Description
/api/widget/token POST Get a session token for widget initialization
/api/widget/services GET Get available services for the business
/api/widget/staff GET Get staff members and their availability
/api/widget/availability GET Check availability for a specific service and date
/api/widget/booking POST Create a new booking

FAQ

Can I use multiple widgets on the same page?

Yes, you can have multiple widgets on the same page by using different container IDs. Make sure each widget has a unique container ID and targets different services or staff members.

Does the widget work on mobile devices?

Yes, the widget is fully responsive and optimized for all devices including mobile phones and tablets. We use modern responsive design principles to ensure a consistent experience across all screen sizes.

Can I customize the booking flow?

Yes, you can customize the booking flow through your dashboard settings. Options include: single-page or multi-step flow, custom form fields, payment requirements, and confirmation settings.

How are bookings confirmed?

When a customer makes a booking through the widget, both the customer and your business receive instant notifications (email and/or SMS). All bookings appear in your dashboard for easy management.

Can I integrate with my Google Calendar or other calendars?

Yes, BookingPro offers integrations with Google Calendar, Outlook, iCal, and other calendar systems. These can be configured in your dashboard's integration settings.
